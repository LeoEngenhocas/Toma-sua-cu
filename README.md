# Site de Zoeira - Gato do Dedo

Um site simples e engraçado com um gato mostrando o dedo do meio.

## Como usar

Abra o arquivo `index.html` no seu navegador web.

Ou, para executar localmente com um servidor:

1. Instale Python (se não tiver).
2. Execute: `python -m http.server 8000`
3. Abra http://localhost:8000 no navegador.

## Conteúdo

- Uma pergunta filosófica: "Qual é a resposta para tudo na vida?"
- A resposta: Um gato mostrando o dedo do meio.
